Hệ thống Quản lý Link Affiliate & Hoa hồng

Next.js 14 · Firebase · TailwindCSS · Recharts

Phiên bản: v1.0 • Ngày: 06/2026 • Đơn vị: Softech Aptech

Ứng dụng web quản lý link affiliate, theo dõi click/conversion và tự động tính hoa hồng. Hệ thống phát triển từ phiên bản cá nhân lên nền tảng đa vai trò với cộng tác viên, dashboard phân tích và landing page.

Phạm vi — 4 module triển khai tuần tự

D · CORE

Quản lý link & hoa hồng

CRUD link, short-link tự động, redirect tracking /r/[slug] → 302, ghi click/conversion, tự tính hoa hồng, chiến dịch, dashboard cơ bản.

A

Đăng ký CTV

Form đăng ký public, admin duyệt/từ chối, khu vực CTV riêng, tạo link, thống kê cá nhân, quản lý payout & lịch sử thanh toán.

B

Dashboard tracking

KPI cards, biểu đồ Recharts (line/bar/top10/funnel), bảng click & conversion, lọc đa chiều, dedupe click, phân quyền dữ liệu tự động.

C

Landing page

Trang sản phẩm SSR/ISR chuẩn SEO, storefront riêng theo CTV (/u/[id] hoặc ?ref=), form thu lead, quản lý sản phẩm, tối ưu hình ảnh.
