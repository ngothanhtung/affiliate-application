Affiliate Link &
Commission Manager

Công cụ web quản lý link affiliate, theo dõi click/conversion và tính hoa hồng — từ phiên bản cá nhân (single-tenant) tới nền tảng đa vai trò có cộng tác viên, dashboard và landing page.

Next.js 14 App Router Firebase Auth · Firestore TailwindCSS Recharts

1 · Tổng quan

App web cá nhân để quản lý link affiliate và theo dõi hoa hồng. Người dùng tạo link tiếp thị, gắn vào chiến dịch, hệ thống tự sinh short-link redirect, đếm click và ghi nhận conversion, từ đó tính hoa hồng kỳ vọng và thực nhận. Khi mở rộng, app chuyển sang nền tảng đa vai trò có cộng tác viên.

Mục tiêu

Tập trung mọi link, biết link nào ra tiền, tính hoa hồng tự động, báo cáo theo tháng/chiến dịch.

Vai trò

Admin/Owner duyệt CTV & set hoa hồng. Affiliate lấy link riêng. Visitor bấm link không cần login.

Ngoài phạm vi v1

Tự crawl hoa hồng từ nền tảng, đối soát thanh toán tự động với nhà cung cấp.

MODULE D

Module D — Quản lý link & hoa hồng

Xác thực & Quản lý Link (CRUD)
Đăng nhập Google / email-password (Firebase Auth), bảo vệ route dashboard. Mỗi link gồm: tiêu đề, link gốc, slug tự sinh, nền tảng (Shopee/Lazada/Tiki/Accesstrade), chiến dịch, % hoa hồng, giá sản phẩm, trạng thái.
Short-link Redirect & Tracking
Route /r/[slug] ghi 1 bản ghi click rồi redirect 302 sang link gốc. Mỗi click lưu referrer, userAgent, country (best-effort). Dedupe theo IP+UA trong cửa sổ ngắn.
Conversion & Hoa hồng
Nhập tay hoặc import CSV (đối soát Accesstrade/Shopee). Tự tính commissionAmount = orderValue × commissionRate nếu không nhập tay. Trạng thái: pending / confirmed / rejected.
Chiến dịch & Dashboard
Gom nhóm nhiều link thành chiến dịch (goal, thời gian). Dashboard: tổng click/conversion, CR%, hoa hồng pending vs confirmed, biểu đồ click theo ngày, hoa hồng theo nền tảng, top link theo doanh thu.

3 · Mô hình dữ liệu (Firestore)

users/{uid}
            links/{linkId} { title, originalUrl, slug, platform, campaignId,
            commissionRate, productPrice, status, clickCount,
            createdAt, updatedAt }
            clicks/{clickId} { linkId, timestamp, referrer, userAgent, country }
            conversions/{convId} { linkId, orderValue, commissionAmount, status, date }
            campaigns/{campId} { name, description, startDate, endDate, goal }

            slugIndex/{slug} { uid, linkId } // tra cứu nhanh khi redirect

slugIndex đặt ở top-level để route /r/[slug] resolve được mà không cần biết uid trước.

MODULE A

Trang đăng ký Affiliate cho CTV

Cho phép cộng tác viên tự đăng ký, admin duyệt, CTV nhận link tiếp thị riêng để kiếm hoa hồng.

Đăng ký CTV (public)

Form: họ tên, email, SĐT, kênh quảng bá, lý do. Tạo Auth + affiliates/{uid} status=pending, gửi email xác nhận.

Duyệt CTV (admin)

Danh sách đơn, lọc theo status, duyệt/từ chối kèm lý do, gửi thông báo kết quả.

Khu vực CTV

Xem sản phẩm khả dụng + % hoa hồng, tạo link riêng (slug gắn affiliateId), thống kê cá nhân, hồ sơ nhận tiền.

Hoa hồng & Payout (admin)

Xem hoa hồng từng CTV theo kỳ, đánh dấu thanh toán (payouts), lịch sử thanh toán.

affiliates/{uid} { name, email, phone, channel, note, status, payoutInfo, ... }
            products/{productId} { name, description, imageUrl, price, commissionRate, status }
            referralLinks/{id} { affiliateId, productId, slug, clickCount, createdAt }
            payouts/{id} { affiliateId, period, amount, status, paidAt }
MODULE B

Dashboard Tracking Click/Conversion

Trung tâm phân tích: admin thấy toàn hệ thống, CTV chỉ thấy dữ liệu của mình.

KPI cards

Tổng click, conversion, CR%, hoa hồng (pending/confirmed/đã trả), EPC (earning per click), so sánh % kỳ trước.

Biểu đồ

Line click/conversion theo ngày, bar hoa hồng theo nền tảng/sản phẩm, top 10 link/CTV, funnel Click→Conversion→Confirmed.

Bảng dữ liệu

Click chi tiết (thời gian, referrer, thiết bị, quốc gia) + conversion, sắp xếp/phân trang/lọc.

Tracking engine

Ghi click async + redirect 302, parse thiết bị/OS/browser, dedupe (IP+UA, 30'), tăng clickCount bằng increment().

clicks/{id} { slug, linkId, affiliateId, timestamp, referrer,
            device, os, browser, country, ipHash }
            conversions/{id} { linkId, affiliateId, productId, orderValue,
            commissionAmount, status, date }

ipHash = hash(IP) để dedupe mà không lưu IP thô — bảo vệ quyền riêng tư.

MODULE C

Landing Page giới thiệu sản phẩm + Link

Trang public đẹp, tải nhanh, SEO tốt. Mỗi CTV có landing/storefront riêng theo affiliateId.

Trang sản phẩm

Ảnh, tên, mô tả, giá, CTA → /r/[slug] track click rồi redirect. SEO: meta + OG image.

Landing theo CTV

/p/[slug]?ref=[affiliateId] hoặc /u/[affiliateId] — gán hoa hồng đúng CTV, storefront mini nhiều sản phẩm.

Tùy biến nội dung

Admin tạo/sửa sản phẩm (ảnh qua Storage, rich text, badge KM). CTV chọn sản phẩm hiển thị.

Thu lead (optional)

Form nhận ưu đãi (email/SĐT) → lưu leads cho remarketing thủ công.

Landing render SSR/ISR cho SEO & tốc độ, ảnh tối ưu qua next/image, mục tiêu Lighthouse Performance > 90.
